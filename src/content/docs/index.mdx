## Mode d’emploi
- Chaque activité a un code (`A01`, `A02`, …), une **durée**, un **niveau** et des **tags**.
- Le bouton **Rechercher** (loupe) permet de filtrer rapidement par mot-clé.
- Les consignes et livrables sont dans la page de l’activité.

## Dernières infos
- Pensez à vérifier la durée et les prérequis avant une séance.
